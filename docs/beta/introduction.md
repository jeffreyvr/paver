# Introduction

Paver is a framework agnostic block based editor. This editor has been made with PHP developers in mind. This means that you can make custom blocks and block options through PHP classes.

The editor itself has been made with SortableJS and plain Javascript, and uses Alpine.js to a great extend.

Dedicated packages for this editor have been made for:
- [Laravel](/docs/beta/laravel-installation)
- [WordPress](/docs/beta/wordpress-installation)

## Pre release

Paver is currently in pre-release, which means the API is subject is subject to change. You can use Paver, but be aware that breaking changes might occur without this being reflected in versioning. Once Paver is stable it will follow semantic versioning.
